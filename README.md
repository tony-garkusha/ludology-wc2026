# Ludology WC 2026 — гонка прогнозистов

Откройте `index.html` в браузере. Страница содержит встроенные проверенные результаты турнира.

Над анимацией доступны два вида:

- **График** — линии показывают изменение очков после каждого матча.
- **Параллельная гонка** — участники движутся вверх и плавно перестраиваются слева направо от лидера к отстающему. При равенстве очков сохраняется порядок предыдущего этапа.

Страница автоматически добавляет стартовую точку, где у всех участников `0` очков. Добавлять отдельную нулевую колонку в Excel не нужно.

Аббревиатуры стран из строки матчей отображаются как emoji-флаги. Неизвестные аббревиатуры автоматически остаются текстом.

Чтобы обновить гонку по ходу турнира, обновите Excel и перегенерируйте встроенные данные:

```bash
/Users/agarkusha/.cache/codex-runtimes/codex-primary-runtime/dependencies/python/bin/python3 generate_tournament_data.py
```

Скрипт читает файл `/Users/agarkusha/Library/CloudStorage/OneDrive-Personal/Ludology WC2026.xlsx`, обновляет `tournament-data.js` и автоматически меняет asset version в `index.html`, чтобы браузеры не держали старые данные в кеше. После этого закоммитьте и запушьте изменения.
