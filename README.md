# Ludology WC 2026 — гонка прогнозистов

Откройте `index.html` в браузере. Страница уже содержит результаты из текущей версии `Ludology WC2026.xlsx`.

Над анимацией доступны два вида:

- **График** — линии показывают изменение очков после каждого матча.
- **Параллельная гонка** — участники движутся вверх и плавно перестраиваются слева направо от лидера к отстающему. При равенстве очков сохраняется порядок предыдущего этапа.

Страница автоматически добавляет стартовую точку, где у всех участников `0` очков. Добавлять отдельную нулевую колонку в Excel не нужно.

Чтобы обновить гонку по ходу турнира:

1. Добавьте название нового матча в строку 2 Excel, начиная с колонки D.
2. Заполните очки за матч в верхней таблице и накопительные суммы в таблице с 12-й строки.
3. На странице нажмите **Загрузить Excel** и выберите обновлённый файл.

Загруженные данные сохраняются в браузере. Для чтения нового Excel странице нужен доступ к интернету, поскольку библиотека чтения `.xlsx` загружается с CDN.
